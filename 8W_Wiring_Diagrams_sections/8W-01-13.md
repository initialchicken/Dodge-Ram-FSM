# Service Procedures (Continued) - Terminal Replacement and Wire Repair

![Diagram](./8W-01-13.jpg)

**Notes:** This page contains service procedures for wire harness repair, not a wiring diagram. Includes instructions for: (1) Stagger cutting wires for repair, (2) Soldering connections using rosin core solder, (3) Applying heat shrink tubing, (4) Re-taping harnesses, (5) Terminal replacement procedures using special tools (Kit 6880, 968W-145, 968W-236, 968W-237), (6) Connector locking wedge tab removal (928W-143). Figure 18 shows stagger cuts on harness wires. Figures 19-21 show terminal removal using special tools and connector locking wedge tabs. Instructions emphasize using rosin core solder only (not acid core) and proper heat shrink tubing application.
