# GENERAL INFORMATION - SERVICE PROCEDURES (Continued) AND SPECIAL TOOLS

![Diagram](./8W-01-14.jpg)

**Notes:** This page contains service procedures for wire repair and diode replacement, along with special tools used for wiring/terminal work. It is an informational page rather than a wiring diagram. Procedures include heat shrink tubing installation over joints, connector repair, wire harness stripping guidelines, and diode replacement steps with current flow direction identification. Special tools shown include Probing Tool Package 6807, Terminal Pick 6680, Terminal Removing Tool 6932, and Terminal Removing Tool 6934. Figure 22 shows diode identification with band around diode indicating current flow direction.
