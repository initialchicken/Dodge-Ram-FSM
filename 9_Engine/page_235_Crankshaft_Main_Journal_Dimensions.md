Thrust journals can be ground in the same increments and using the same specifications as all other main journals. The main journal radius may be ground using either the preferred or the alternative procedure providing the thrust surface width is not being ground. The preferred procedure must be used when the main bearing thrust width surface is ground. When the thrust surface width requires grinding, the main journal must be ground to the same undersize dimension (Fig. 33).
